## async、await ⭐

[阮一峰-async 函数](https://es6.ruanyifeng.com/#docs/async)

```js
// 异步循环打印
function sleep(time, val) {
  return new Promise(function(resolve, reject) {
    setTimeout(() => {
      resolve(val)
    }, time)
  })
}

var run = async function() {
  for (var i = 0; i < 10; i++) {
    // await后函数挂起，结束后继续执行
    var result = await sleep(1000, i)
    console.log(result)
  }
}

run()
```
