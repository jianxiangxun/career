# cocoapods

pod 使用小记

cocoapods 替换镜像源，解决安装缓慢、不成功问题： https://mirrors.tuna.tsinghua.edu.cn/help/CocoaPods/

卸载： https://www.jianshu.com/p/da85db9d9523

安装： sudo gem install cocoapods

安装依赖包包: 发生执行 pod install 或 pod update 都卡在 Analyzing dependencies 的情况，是因为要升级 CocoaPods 的 spec 仓库，命令后添加--verbose --no-repo-update 参数可以省略此步。
